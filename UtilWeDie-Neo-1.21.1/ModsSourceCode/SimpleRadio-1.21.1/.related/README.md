# Overview

This is the folder where information for any Simple Radio related project can be hosted, and serves to make the process of displaying and updating related projects easier. 

For now, it will be used so icons can be updated simultaneously across project pages and independently of me.
Later on, dynamic pages will be generated on the Simple Radio website using the information
contained in these folders.

Only the icon is required, but if you would like the project to also show up on the website (when it's added),
you may include other information here. A project will not be loaded in the website if it does not have a **description.md**.

### Key Files
| Filename                          | Description                                                                                           |
|-----------------------------------|-------------------------------------------------------------------------------------------------------|
| icon.png                          | The main icon for the project, can be taken from Modrinth; should be **96x96** pixels.                |
| icon_large.png                    | The large icon for the project, will be displayed in the dynamic pages; should be **256x256** pixels. |
| description.md                    | The description for the project in markdown, will be displayed in the dynamic pages.                  |
