## 本轮改动

- 在背包和箱子等物品界面内按资产管理快捷键，也能打开资产管理器。
- 状态方块改用中文：`标`、`管`、`材`、`产`、`掉`、`同`、`!`。
- 列表右上角显示“当前条目范围 / 总数”，内容超出一页时显示滚动条。
- 世界内打开、无背景虚化、人工标注和导出逻辑保持不变。

## 必测（约 5 分钟）

- [ ] **T1 世界内打开**：站在世界中按资产管理快捷键。预期正常打开，界面不虚化。
- [ ] **T2 背包内打开**：先打开玩家背包，再按资产管理快捷键。预期打开资产管理器；按 `Esc` 后回到刚才的背包。
- [ ] **T3 箱子内打开**：先打开箱子，再按资产管理快捷键。预期打开资产管理器，列表能包含箱子当前可见物品；按 `Esc` 后回到同一个箱子界面。
- [ ] **T4 不干扰文字输入**：打开聊天框并输入小写或大写字母 `o`。预期只是正常输入文字，不会打开资产管理器。
- [ ] **T5 已标注列表**：清空搜索词，点击左侧“人工标注”。预期右上角显示总数；将鼠标放在中间列表滚动，能够看到当前保存的全部 10 条人工标注，并能看到右侧滚动条移动。
- [ ] **T6 中文状态**：任意选择一条记录。预期不再显示 `H C I O L S`；显示下列中文状态，右侧检查器也使用中文名称。

## 状态方块含义

- `标`：由你在游戏内人工标注。
- `管`：已经进入项目资产目录。
- `材`：被某个配方当作材料。
- `产`：被某个配方作为产物。
- `掉`：已参与 Loot / 战利品管理。
- `同`：游戏端与桌面目录状态一致。
- `!`：记录存在异常或需要处理。
